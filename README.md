# Flow Field Dashes

A small generative canvas study inspired by rotating dash-field animation references.

The artwork keeps each dash fixed on a grid while its angle is driven by a moving mathematical field. The result is a looping surface of strokes that can shift between flow fields, radial spokes, concentric ripples, traveling waves, vortex patterns, and boat-wake motion.

## Run

Open `index.html` in a browser.

## Controls

- `Pause` freezes the animation.
- `Randomize` changes the field seed.
- `Export PNG` saves the current frame.
- `Export SVG` saves the current frame as editable vector lines.
- `Save`, `Load`, and `Delete` manage named presets in browser storage.
- `Share URL` copies a link containing the current configuration.
- `Pattern` switches between flow, radial, ring, wave, vortex, and boat-wake formulas.
- `Speed`, `Organic easing`, `Field pull`, `Wave scale`, and `Grid density` tune the motion and composition with either sliders or exact numeric inputs.
